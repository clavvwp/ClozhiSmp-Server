# Audio Folder

Taruh file audio kamu di sini dengan nama:

  sonata-moonlit.mp3

Cara mendapatkan file audio:
1. Download "Sonata Moonlit" dari YouTube atau platform musik
2. Convert ke .mp3 (pakai https://cloudconvert.com jika perlu)
3. Rename jadi: sonata-moonlit.mp3
4. Taruh di folder ini: /public/audio/sonata-moonlit.mp3

Format yang didukung browser: .mp3, .ogg, .wav
